# Solutions Portal

Welcome to the internal solutions knowledge base.

Use the **search bar** to find solutions by technology or keyword,
or browse by topic on the **Tags** page.

## How to add your solution

1. Create a GitHub repo named `sol-<your-solution-name>`
2. Add the topic `solution-doc` plus any domain topics (e.g. `payments`, `auth`)
3. Write your solution in the repo's `README.md`
4. The portal rebuilds automatically every 6 hours
